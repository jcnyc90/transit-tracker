HOW TO DEPLOY

This app needs a free Vercel backend to proxy MTA's API (MTA blocks direct browser requests). Steps:

  1. Install Vercel CLI: npm i -g vercel
  2. In the project folder, run: vercel
  3. Add env var: MTA_BUS_KEY = cae896f4-...
  4. Redeploy: vercel --prod
  5. Share the .vercel.app URL with your partner
