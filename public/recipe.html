ONE-TAP SETUP

Once set up, tap Share → Recipe Chat directly from MealBoard — no steps in between.

1Open the Shortcuts app and tap + to create a new shortcut.
2Tap Add Action → search Receive → add Receive Input from Share Sheet. Set type to PDFs.
3Add action: search Encode → add Encode/Decode. Set to Base64 Encode. Connect input to Shortcut Input.
4Add action: Text. Type your site URL then # then insert the Encoded Text variable. Example: https://yoursite.vercel.app/recipe# + variable.
5Add action: Open URLs. Connect it to the Text above.
6Tap the shortcut name, rename it Recipe Chat. In settings (⋯), enable Show in Share Sheet.
Works best with PDFs under 1.5 MB. Larger files: use the "Choose PDF" button above instead.
